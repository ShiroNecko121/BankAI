How the ANN Model Works

BankAI utilizes a Supervised Artificial Neural Network (ANN) trained on historical telemarketing data to predict if a customer will subscribe to a term deposit. It analyzes 8 key demographic and historical features to calculate a probability score.

The Explainable AI (XAI) module then processes the input data against standard financial logic to generate human-readable reasons explaining why the model made its decision.
